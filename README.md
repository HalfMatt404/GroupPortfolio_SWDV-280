
<!---------SCSS Instructions-------->
# Install Sass
```bash
npm install -g sass
```

# Watch SCSS
```bash
sass --watch assets/scss:assets/css
```

# Stop Watching 
Press Ctrl + C

# Compile SCSS once
```bash
sass assets/scss/main.scss assets/css/main.css
```

# Important Notes
- Only edit files inside `assets/scss/`
- Never manually edit files inside `assets/css/`
- Always run Sass commands from the project root directory
- If using watch mode, stop it before closing your terminal
